App Privacy Information for MyWaterSip: Reminder & Tracker

This document provides a comprehensive overview of the privacy practices for the MyWaterSip: Reminder & Tracker app, to be used when submitting privacy information to App Store Connect.

Data Types Collected

1. Health & Fitness Data

  • Water intake tracking: The app does not collect user-inputted data about daily water consumption.
  • Usage: Data is used only within the app to display statistics, track achievements, and show progress.
  • Storage: Stored locally on the device using AsyncStorage.
  • Sharing: Not shared with third parties.
  • Tracking: Not used for tracking purposes.
  • Data Linked to User: Not linked to the user's identity.

2. User Content

  • Personal goals: User-defined water intake goals.
  • Reminders: User-configured reminder times.
  • Usage: Used only to provide app functionality.
  • Storage: Stored locally on the device.
  • Sharing: Not shared with third parties.
  • Tracking: Not used for tracking purposes.
  • Data Linked to User: Not linked to the user's identity.

3. Purchase Data

  • Subscription information: Data about user's subscription status.
  • Usage: Used to unlock premium features.
  • Processing: Processed through Apple's App Store services.
  • Storage: Subscription receipts stored locally for validation.
  • Sharing: Only shared with Apple for processing payments.
  • Tracking: Not used for tracking purposes.
  • Data Linked to User: Linked to App Store account, managed by Apple.

4. Identifiers

  • Device identifiers: Used only for managing notifications.
  • Usage: To schedule and deliver water reminders.
  • Storage: Not stored permanently.
  • Sharing: Not shared with third parties.
  • Tracking: Not used for tracking purposes.
  • Data Linked to User: Not linked to the user's identity.

Data Not Collected

The app does NOT collect or access the following:

  • Location data
  • Contacts
  • Photos/Videos
  • Browsing history
  • Search history
  • User ID (except Apple ID for purchases, managed by Apple)
  • Diagnostics
  • Precise location

Third-Party Services

  • Apple In-App Purchase: For managing subscriptions.
  • Notifications: For delivering scheduled water reminders.

Data Encryption

All locally stored data is encrypted using the device's built-in security features.

Data Access

Users can delete all their data by uninstalling the app. No data is stored on external servers.

Data Retention

All user data is stored locally on the device and can be deleted at any time by uninstalling the app.

Children's Privacy

The app does not knowingly collect data from children under the age of 13.

Changes to This Privacy Information

This document may be updated as the app evolves. The latest version will be submitted with each app update.
